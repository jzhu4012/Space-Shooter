## Overview

The first out of five projects from the video ["Master Python by making 5 games"](https://youtu.be/8OMghdHP-zs?si=4elZ6K6DPKeONA2O "The new ultimate introduction to pygame").
This project discusses the basics of pygame, such as sufaces, sprites, events, and time. It is a basic shooter game based off the classic Asteroids arcade game.
Use WASD to move around and \[SPACE\] to shoot.


## Personal Commentary
